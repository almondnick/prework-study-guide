HTML

  • HTML stands for Hypertext Markup Language.
  • HTML is used to give the content structure.
  • The title element defines the title for a webpage. This element is extremely important for Search Engine Optimization (SEO).
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The style element defines the CSS styles associated with a website.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata such as character set, description, keywords, author, and viewport.
  • The script element embeds client-side scripts in the HTML document.
  • h1, h2, h3, etc. elements represent the level of heading a given text block represents.
  • The p element represents a paragraph or block of text.
  • The ul, ol, and li elements represent unordered lists, ordered lists, and list items.

CSS

  • CSS stands for Cascading Style Sheets.
  • CSS is used to style a webpage (colors, aesthetics, visual layout, and organization).
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside of an element.
  • The CSS file is linked with the "head" section of the HTML file.

Git

  • git add -A: stages the changes made.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git commit -m "comment": message with commit to describe changes made.
  • git pull origin main: pull base branch into feature branch before push.
  • git push origin branch-name: push up changes.
  • git checkout: switch between branches.

JavaScript

  • JavaScript handles the logic of the website and allows users to interact.
  • A variable is a named container that allows us to store data in our code.
  • The computer flows through if and else if statements to see if they are truthy or falsy and performs an action based on the statement.
  • An array is a single variable that is used to hold a group of data.
  • Items in an array are numbered starting with 0.
  • for loops are used to perform a task on all items in an array by allowing a single code block to be executed over and over.
  • functions are a set of instructions telling the computer how to perform a certain task.
  • For a function to work, it must be called to do so. Call a function by writing "functionName"().
  • Control flow is the order in which a computer executes code in a script.
